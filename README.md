# chip8.java

Chip8 emulator rewrite in java

## Build
```sh
javac Pixels.java
javac Chip8.java
javac Chip8Runner.java
javac Chip8Tester.java
```

## Run

Run normally
```sh
java Chip8Runner
```

Run tests
```sh
java Chip8Tester
```
