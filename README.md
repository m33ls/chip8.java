# chip8.java

Chip8 emulator rewrite in Java

## Build
```sh
javac *.java
```

## Run

Run normally
```sh
java Chip8Runner <Path to ROM.ch8>
```
for example:
```sh
java Chip8Runner ibm-logo.ch8
```

Run tests
```sh
java Chip8Tester
```
